Citations

Sources and tools used to design and build Numbl.

Website Design

https://fonts.googleapis.com

Font library used to generate and create fonts for headings, subheadings, and paragraph variations.

https://boxicons.com/

Resource for icons in place of designs. Used throughout our project to draw users’ attention to important details.

https://color.adobe.com/

A tool used to brainstorm color schemes for themes throughout our website.

https://rawpixel.com/

Rawpixel is a visual design archive that provides users with Creative Commons Zero (CC0) images and designs which we used in our website to draw in viewers' attention.

Content

https://render.com/

Render is a hosting site that allows users to deploy sites. Our project is hosted on Render. In addition to hosting, Render provides Cloudflare services such as DNS, SSL, and other protections.

https://www.khanacademy.org/

Khan Academy is an educational website with free courses. We used it as a resource while creating our own course flow and as inspiration for layout ideas.

https://www.flippedmath.com/

FlippedMath is an online resource we used as a guide while organizing course topics like Algebra and Geometry.

http://www.wallace.ccfaculty.org/book/book.html

Tyler Wallace’s "Beginning and Intermediate Algebra" provides open-access lessons and practice problems for Algebra 1 and Algebra 2. This resource was used to guide the structure and content of our topic notes. Creative Commons Attribution 3.0 Unported (CC BY 3.0)

https://www.youtube.com/@TheAlgebros

The Algebros (YouTube channel) — Math review videos used as embedded “Unit Review” videos on the math course pages. Content created and published by The Algebros.
