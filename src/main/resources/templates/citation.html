Additions

https://render.com/

Render is a hosting site that allows users to access servicers that can be used to deploy sites. Our project is hosted on Render. In addition to hosting servers, Render provides users with CloudFlare which is an industry-leading security service which offers DNS, WAF and SSL encryption, protecting our users from data breaches.

https://www.khanacademy.org/

Khan Academy is an educational website that offers free courses on a variety of subjects. We used Khan Academy as a resource to help us create our own courses and lessons for our website. We also used Khan Academy as a source of inspiration for our website design and layout.

https://www.flippedmath.com/

Flipped math is an online resource that we used throughout our project as a guide through course. They offer specialized courses on topics like Algebra and Geometry which is our website's main focus.
